# Package index

## All functions

- [`dst_get_all_data()`](https://ropengov.github.io/dkstat/reference/dst_get_all_data.md)
  : This function returns the entire table requested data from the
  StatBank. (http://www.statistikbanken.dk/statbank5a/ or
  http://www.dst.dk)
- [`dst_get_data()`](https://ropengov.github.io/dkstat/reference/dst_get_data.md)
  : This function returns the requested data from the StatBank table.
  (http://www.statistikbanken.dk/statbank5a/ or http://www.dst.dk)
- [`dst_get_tables()`](https://ropengov.github.io/dkstat/reference/dst_get_tables.md)
  : This functions downloads all the available tables.
- [`dst_meta()`](https://ropengov.github.io/dkstat/reference/dst_meta.md)
  : This function returns meta data for a table from StatBank Denmark /
  Statistics Denmark. (http://www.statistikbanken.dk/statbank5a/ or
  http://www.dst.dk)
- [`dst_search()`](https://ropengov.github.io/dkstat/reference/dst_search.md)
  : This function provides a search function for the description field
  of available data series
- [`tables`](https://ropengov.github.io/dkstat/reference/tables.md) :
  Cached list of all available tables
